## Tips

### How to use Git

+ Clone repo:
```bash
cd path/to/work/dir
git clone https://https://github.com/rika7596/python-study.git
```

+ Add and commit changes
```bash
cd path/to/work/dir/python-study
git add --all
git commit -m "Commit message"
git push
```

+ Pull new commits
```bash
cd path/to/work/dir/python-study
git pull
```
